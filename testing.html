Testing

Optimisation

  • I have ensured to use as few images as possible so that the loading times are minimal.
  • There are a few images that take some time to load due to the large pixel density, however, I have sacrificed loading times for a higher quality image in these cases, as I feel this is more important to the quality of the website.
  • I have not embedded anything apart from images which are stored locally, which speeds up loading time significantly.
  • Any external links are succinct and descriptive.

Debugging

  • I have made use of the official CSS and HTML validators by w3, which have been linked in my legal and security page.
  • I tested every single HTML file and while, there were initially issues with there being unnecessary ARIA elements, I got rid of them and this fixed those errors leaving me with each HTML file having no errors or warnings.
  • I also tested every single CSS file and had no errors or warnings and each file was compliant with the CSS level 3 + SVG standard which can also be seen in my legal and security page.

Browser testing

  • I have tested my website on both a chromium and quantum based browser using both a desktop and mobile view, to ensure that it runs perfectly no matter how the user views it.
  • I have not shown every single page as it would be redundant, but as seen in the screenshots, both browser engines display the website as intended with no side effects.

Accessibility

  • To ensure that My website is accessible, I have used the WAVE web extension to test each webpage on my website.
  • This can be seen in more detail in my accessibility section, but as you can see by the images, there are no Errors and only a couple of warnings which are due to elements which do not have much effect on accessibility.
  • These warnings are mainly due to redundant links which are due to references, or elements that appear to look like headings such as my logo, or image alt text where in the quiz, the alt text had to be ambiguous.
  • Overall, as seen in the evaluation of my website, it complies with the WCAG level AA standard which ensures that my website is accessible.
